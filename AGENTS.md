# AGENTS.md

## Purpose

Maintain this repository as a concise, evidence-based guide for learning about investing.

## Rules

- Address the user as TSAO in every response.
- Prefer primary research, regulators, official statistics, and index-provider methodology or scorecards.
- Support every quantitative or empirical claim with a directly linked source.
- Record the source's publication date and the date accessed when adding time-sensitive data.
- Distinguish facts, calculations, assumptions, and opinions explicitly.
- Recalculate examples independently and verify units, dates, currencies, fees, and tax assumptions.
- Never fabricate returns, citations, fund characteristics, or certainty.
- State confidence as high, moderate, low, or unknown for substantive conclusions.
- Do not call a product or strategy “best” without defining the investor, objective, horizon, constraints, costs, taxes, benchmark, and risk measure.
- Do not infer that past performance will persist.
- Keep examples jurisdiction-neutral unless the jurisdiction is specified.
- Prefer simple portfolios and explanations. Every section, parameter, and abstraction must have a clear job.
- Use `gh` for GitHub operations and include full GitHub URLs in references.

## Verification before committing

1. Open every new source and confirm that it supports the adjacent claim.
2. Check whether newer data supersedes any dated figure.
3. Test all calculations independently.
4. Check links and Markdown formatting.
5. Review `git diff` and ensure no credentials or personal financial data are present.
